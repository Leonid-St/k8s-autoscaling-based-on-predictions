# Quota Manager API, gRPC: QuotaLimitService

A set of methods for managing quota limits.

## Methods

#|
||Method | Description ||
|| [Get](get.md) | Returns the specified quota limit. ||
|| [List](list.md) | Retrieves the list of quota limits for a given service. ||

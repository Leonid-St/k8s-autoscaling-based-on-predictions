## Удаление приватного соединения {#priv-del}

Чтобы удалить cуществующее приватное соединение из транка, создайте [новое обращение в поддержку]({{ link-console-support }}).

### Обращение в поддержку для удаления приватного соединения из транка {#priv-ticket}

Обращение должно быть оформлено следующим образом:

```s
Тема: [CIC] Удалить приватное соединение из транка.

Текст обращения:
Прошу удалить приватное соединение из транка.

trunk_id: euus5dfgchu23b******
prc_id: cf3qdug4fsf737******
```

где:

* `trunk_id` — идентификатор транка, в котором создано приватное соединение.
* `prc_id` — идентификатор приватного соединения, которое нужно удалить.

### Ответ поддержки по обращению клиента {#priv-ticket-resp}

После завершения работ по удалению приватного соединения поддержка уведомит вас.

Пример ответа поддержки:

```s
Приватное соединенение cf3qdug4fsf737****** успешно удалено
из транка euus5dfgchu23b******.
```
